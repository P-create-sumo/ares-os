# ARES-OS — Field Testing Guide

## Requirements
- Any modern browser (Chrome, Firefox, Edge)
- No installation required
- Works offline (Progressive Web App — roadmap)

## Live Demo
Open: https://ares-os-app-7bb72773.base44.app/UIDemo

## What to test
1. **Pre-flight wizard** — run through the diagnostic checklist
2. **Shared interface** — open on two devices simultaneously (operator + navigator view)
3. **Battery tracker** — log a pack, simulate charge cycles
4. **Mission debrief** — log a sortie outcome
5. **Analytics** — check the dashboard after 3+ sorties logged
